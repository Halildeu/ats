# G0 Sektör Paketi — Holding-İçi İlk Design-Partner'lar (inşaat / turizm / telekom)

> Owner kararı: G0'a **holding-içi (warm) kurumlarla** başla. Bu doküman 3 sektör için tailored pitch + discovery + DPO açısı + golden-fixture fırsatı verir. **Outreach/pilot yürütmesi owner işidir**; bu paket onu hızlandırır.

## Neden holding-içi başlamak doğru (de-risk)
| Kazanım | Nasıl |
|---|---|
| Gerçek mülakat verisi | holding kurumlarının gerçek mülakatları → **golden Türkçe fixture** (M6 kriter 6) tek hamlede |
| Gerçek DPO süreci | holding DPO/hukuk → procurement template pack provası + yazılı sign-off (M6 kriter 4) |
| Gerçek ATS | kurumların mevcut ATS'i → export/write-back yolu doğrulama (M6 kriter 3) |
| Hız + güven | iç erişim → haftalar yerine günler; pilot riski düşük |

## ⚠️ Friendly/non-friendly dengesi (M6 disiplini)
Holding kurumları **friendly** sayılır. Tam **G0=GO** için (turnkey pack kriter 2):
- Holding kurumları → **ilk pilotlar + golden fixture + DPO provası** (≥1-2 ücretli iç-pilot OK).
- **+ ≥2 non-friendly (dış) ücretli pilot** → "satılabilirlik" kanıtı (iç-kullanım ≠ pazar kanıtı).
- Strateji: holding ile **kanıt + olgunluk** üret → dış kurumlara **referanslı** çık.

## Sektör tailored G0 (inşaat / turizm / telekom)

### 🏗️ İnşaat
- **İşe-alım profili:** saha (şantiye şefi/mühendis/usta) + ofis (proje/satınalma/İK) + yüksek mavi-yaka hacmi + taşeron/proje-bazlı dalgalı.
- **Neden interview-evidence:** çok-lokasyonlu, yüksek-hacim, **denetlenebilirlik** (kamu ihalesi/iş-güvenliği uyumu) → kararın kanıta dayanması değerli.
- **KVKK/DPO açısı:** saha personeli + taşeron veri; kayıt izni + saklama; iş-sağlığı-güvenliği bağlamı.
- **ATS:** çoğu kurumsal İK paketi/lokal ATS → **export taban** muhtemelen yeter (write-back şart değil).
- **Golden fixture:** mühendis/proje pozisyonu paneli (Türkçe, teknik terim zengin → iyi WER/citation testi).

### 🏖️ Turizm
- **İşe-alım profili:** sezonsal **yüksek-hacim** (otel/F&B/resepsiyon/animasyon) + çok-dilli + hızlı toplu işe-alım.
- **Neden interview-evidence:** sezon öncesi yüzlerce mülakat → **tutarlı, denetlenebilir** değerlendirme + işveren-markası (aday deneyimi).
- **KVKK/DPO açısı:** yoğun aday hacmi + kısa süreli kayıt → **retention/silme** kritik; çok-dilli rıza.
- **ATS:** sezonsal hacim için ATS/portal → export + opsiyonel write-back.
- **Golden fixture:** çok-konuşmacı panel + ortam-gürültülü senaryo (gerçekçi DER testi).

### 📡 Telekom
- **İşe-alım profili:** teknik (şebeke/yazılım/saha) + çağrı-merkezi yüksek-hacim + düzenli kadro.
- **Neden interview-evidence:** **regülasyon + denetim** ağırlıklı sektör → audit-ready kanıt + bias-risk yönetimi / tutarlılık kanıtı (kurumsal yönetişim) güçlü satış.
- **KVKK/DPO açısı:** olgun DPO/InfoSec (sektör regülasyonu) → **en iyi procurement-pack provası**; güçlü güvenlik soruları (security-posture-whitepaper burada işe yarar).
- **ATS:** olgun kurumsal ATS (write-back-3-koşul gündeme gelebilir).
- **Golden fixture:** teknik + çağrı-merkezi pozisyon paneli (terminoloji + telefon-sesi varyasyonu).

## Discovery akışı (her kurum için)
1. `g0-one-pager.md` + sektör-pitch (üstteki "neden") → TA/HR + DPO.
2. `g0-icp-questionnaire.md` doldur → ICP rubric skor → `prospect-tracker.csv`.
3. Pilot kapsamı → `g0-loi-template.md` (iç-pilot da yazılı + bedel/süre/metrik).
4. DPO: `g0-turnkey-decision-pack.md` DPO script + `docs/procurement/` pack → yazılı sign-off.
5. Mülakat verisi → `docs/eval/golden-fixture-collection-pack.md` (consent + redaction) → fixture.

## Owner sıradaki adım
- Her sektörden **1-2 kurum** seç (isim) → `prospect-tracker.csv`'ye gir → outreach.
- Hedef: holding'den **2-3 iç-pilot + golden fixture** → sonra **2 dış non-friendly**.
